# finpay-dashboard
FinPay is a modern Razorpay-inspired fintech dashboard built using HTML, CSS, and JavaScript. It includes authentication pages, payments, transactions, customers, analytics, settlements, payouts, responsive UI, export features, interactive modals, and dashboard management with a clean and professional user experience.


# FinPay Dashboard Clone

A modern Razorpay-inspired fintech dashboard UI built using HTML, CSS, and JavaScript.
This project simulates a complete online payment management platform with responsive layouts, interactive dashboards, authentication flows, transaction management, analytics, settlements, payouts, and customer handling.

## Live Demo

https://shirishanambala.github.io/finpay-dashboard/



## Features

* User Authentication (Login & Signup)
* Interactive Dashboard UI
* Payments Management
* Transactions Tracking
* Customers Management
* Analytics Dashboard
* Settlements & Payouts
* Export Functionality
* Responsive Sidebar Navigation
* Interactive Popups & Modals
* Search & Filter Features
* Responsive Design for All Devices
* Modern Fintech UI Inspired by Razorpay



## Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
